# 🛒 CHICOW – Premium Streetwear Brand

Welcome to the official repository of **CHICOW**, a premium streetwear and clothing brand e-commerce platform built with Next.js, TypeScript, and Tailwind CSS.

🌐 **Live Demo:** [chicowclothingbrand.vercel.app](https://chicowclothingbrand.vercel.app)

---

## 🚀 Features

*   **Modern UI/UX:** Clean, responsive, and premium streetwear-focused design.
*   **App Router:** Utilizing Next.js latest App Router structure for efficient routing and layouts.
*   **Component-Driven:** Built using modular and reusable UI components (with shadcn/ui integration).
*   **Fully Responsive:** Optimized for Mobile, Tablet, and Desktop screens.

---

## 🛠️ Tech Stack

*   **Framework:** Next.js (App Router)
*   **Language:** TypeScript
*   **Styling:** Tailwind CSS
*   **Deployment:** Vercel
